# 背景

学生担当嶋崎さんと ICT 推進室の関さんを交えて YCU スケジュールを 2021 年度正式に運用していくにはという話し合いをした。

# 話合い録

二次利用の利用について規定されている。学内の公式ではない場所に転載されてしまうことが二次利用にあたる。

顧問の先生も交えてやらないと無理かもしれない。

責任の所在を明らかにしたい。

今後は顧問の先生と教務の方とで話を進めていく必要がある。

# 現在の課題

- 個人情報の保護について
- 責任の所在について
- 継続性や保守性などについて
