# 概要

Engine[`ˈendʒɪn`]はプログラミングを勉強することを目的とした横浜市立大学の公認サークルです。

# 活動理念

Engine の活動理念は**プログラミングを通じて学生生活を豊かでより充実したものにすること**です。

# 歴史

| :-----------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
